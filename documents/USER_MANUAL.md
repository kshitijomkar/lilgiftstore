# User Manual - The Lil Gift Corner

**Version**: 1.0  
**Audience**: End Users (Customers and Store Admins)  
**Last Updated**: January 2025

---

## For Customers

### Getting Started

1. **Visit Website**: Navigate to the store homepage
2. **Browse Products**: Explore gift collections
3. **Search**: Use search bar to find specific items
4. **Filter**: Apply category and price filters

### Shopping

1. **View Product**: Click on product card
2. **Add to Cart**: Select quantity and click "Add to Cart"
3. **View Cart**: Click cart icon in navigation
4. **Checkout**: Click "Proceed to Checkout"
5. **Payment**: Complete payment via Stripe
6. **Confirmation**: Receive order confirmation

### Account Management

1. **Register**: Click "Sign Up" and fill form
2. **Login**: Use email and password
3. **View Orders**: Access profile to see order history
4. **Wishlist**: Save favorite products for later

---

## For Store Admins

### Admin Dashboard

Access admin panel at `/admin` using admin credentials:
- Email: admin@thelilgiftcorner.com
- Password: Admin@123 (change this!)

### Managing Products

1. **Add Product**: Click "Add Product" button
2. **Edit Product**: Click edit icon on product row
3. **Delete Product**: Click delete icon (with confirmation)
4. **Update Stock**: Edit stock quantity field

### Managing Orders

1. **View Orders**: See all orders in Orders section
2. **Update Status**: Change order status (pending/completed/cancelled)
3. **View Details**: Click on order to see full information
4. **Filter Orders**: Filter by status or date

### Analytics

View dashboard for:
- Total sales
- Number of orders
- Customer count
- Product statistics
