## Welcome!

Thank you for considering contributing to The Lil Gift Corner! We welcome contributions from everyone.

---

## How to Contribute

### Reporting Bugs

1. Check if bug already reported in GitHub Issues
2. Open new issue with:
   - Clear title
   - Description
   - Steps to reproduce
   - Expected vs actual behavior
   - Environment details

### Suggesting Features

1. Check existing feature requests
2. Open new issue with `[Feature Request]` label
3. Describe use case and benefit
4. Discuss with maintainers

### Code Contributions

1. **Fork** the repository
2. **Clone** your fork
3. **Create branch**: `git checkout -b feature/your-feature`
4. **Make changes**
5. **Test** your changes
6. **Commit**: Follow commit message conventions
7. **Push**: `git push origin feature/your-feature`
8. **Pull Request**: Open PR to main branch

---

## Development Guidelines

- Follow coding standards (see Developer Guide)
- Write tests for new features
- Update documentation
- Keep commits focused and atomic
- Be respectful and constructive
