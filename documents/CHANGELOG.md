# Changelog - The Lil Gift Corner

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

---

## [2.0.0] - 2025-01-08

### Added
- Complete modular backend architecture
- Product reviews and ratings system
- Wishlist functionality
- Coupon/discount system
- Order tracking feature
- Admin analytics dashboard
- Sales reporting
- Enhanced SEO with structured data
- Comprehensive documentation suite
- Professional UI components (49 Shadcn components)

### Changed
- Refactored backend into modular structure
- Upgraded React to v19
- Improved database indexing (13 indexes)
- Enhanced error handling
- Optimized API response times

### Security
- Implemented JWT authentication
- Added bcrypt password hashing
- Configured CORS properly
- Input validation with Pydantic

---

## [1.0.0] - 2024-11-01

### Added
- Initial MVP release
- Product catalog
- Shopping cart
- Stripe checkout
- User authentication
- Admin dashboard
- Custom gift requests
- Contact form
