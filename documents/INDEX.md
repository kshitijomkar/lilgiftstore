## Documentation Standards

All documentation in this suite follows these standards:

- **Format**: Markdown (.md)
- **Version Control**: All docs tracked in Git
- **Updates**: Reviewed quarterly
- **Accuracy**: Cross-referenced with codebase
- **Consistency**: Unified terminology (see Glossary)
- **Accessibility**: Clear language, well-structured
